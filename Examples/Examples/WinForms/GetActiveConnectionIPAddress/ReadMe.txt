---------------------------------------------------------------------------

  Get Active Connection IP Addresses WinForms Example

---------------------------------------------------------------------------

Introduction:

  This example demonstrates how to determine the client and server IP addresses
  for an active connection.


Supported OS:

  Windows 2000 and later


Usage:

  Connect to any connection that uses a client and server IP address. Once
  connected, run the application. Select the connection name from the list of
  active connections drop down, and click the Get Address button. The IP 
  addresses will be displayed in their respective boxes.
