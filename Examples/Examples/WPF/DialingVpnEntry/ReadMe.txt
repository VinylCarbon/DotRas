  Dial VPN Entry WPF Example

---------------------------------------------------------------------------

Introduction:

  This example demonstrates how to dial a virtual private network (VPN) entry 
  in a Windows phone book.


Supported OS:

  Windows 2000 and later


Usage:

  Once the application has started, select the connection name from the list
  of available entries in the Connection Name drop down. Once the entry has
  been selected, click the dial button. To disconnect while connecting, click
  the disconnect button.
